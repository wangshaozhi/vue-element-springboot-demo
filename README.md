# vue-element-springboot-demo
vue-element-springboot-demo 
### 项目介绍
+ 前端采用vue+element-ui来实现,对应文件为vue-element-springboot-back-end
+ 后端采用springboot+shiro来实现,对应文件为vue-element-springboot-front-end
+ 数据版本为5.7，数据库框架采用mybatis,对应脚本文件为test.sql
### 主要功能
+ 实现了权限控制
+ 实现了基本后台管理的框架，包含三级菜单
